# vue-project-facture

## Description

Ce projet est une application de gestion de factures que j'ai réalisée en suivant le tutoriel de [John Komarnicki](https://youtu.be/vsJtN54aA7w?si=j21Q9dq066AGf7Yp) dans le cadre de mon apprentissage de Vue.js.

Le projet original utilisait Vuex pour la gestion de l'état. Je l'ai modernisé en le migrant vers Pinia et en mettant à jour les dépendances afin qu'il soit compatible avec les versions récentes suivantes :

* Vue 3.5.32
* Pinia 3.0.4
* Firebase 12.14.0

J'ai apporté quelques modifications visuelles mineures ainsi que des améliorations UX pour offrir une expérience utilisateur plus intuitive, tout en respectant l'esprit du projet original.

L'objectif principal de ce projet était d'approfondir ma compréhension de l'écosystème Vue, de la gestion d'état avec Pinia et de l'intégration de Firebase dans une application moderne.

## Ce que j'ai appris

Au cours de ce projet, j'ai pu :

* Intégrer Firebase dans une application Vue.js.
* Manipuler et synchroniser des données entre Vue.js et Firebase.
* Approfondir ma compréhension de la gestion de l'état avec Pinia après avoir migré un projet utilisant Vuex.
* Mieux comprendre comment organiser et appeler des fonctions afin d'implémenter les opérations CRUD (*Create, Read, Update, Delete*).
